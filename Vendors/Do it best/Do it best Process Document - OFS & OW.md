# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 OFS ORDER RULES**

### **1.1.1 General OFS Rules:**

  - > Out-of-Stock (OFS) orders apply only when the item is out of stock in every warehouse.

  - > Dropship is NOT allowed for OFS items.

  - > The “Future Reserve” checkbox must be selected in the Do It Best portal.

  - > ETA is Not Available.

  - > Always check existing OFS Vendor Orders (VO) before creating a new one.

### **1.1.2 Freight Restrictions:**

Heavy / Freight / Clay items that are OFS may:

  - > Require LTL freight quote, or

  - > Be marked Discontinued if not orderable.

**1.2 OFS PO & NOTES RULES**

For all OFS vendor orders:

**1.2.1 PO Rules:**

  - > Use Store PO only

  - > Do NOT include “DKH-” prefix

**1.2.2 Shipping Address:**

  - > Ship to DK Hardware Warehouse

**1.2.3 Internal Notes Must Include:**

  - > Vendor: Do It Best

  - > Status: OFS

  - > Any Big / Freight restrictions

  - > Future Reserve applied

## **1.3 OTHER WAREHOUSES**

When an item is not available in the Main warehouse but exists in other warehouses, it can be processed through the **Other Warehouse In-House flow**.

### **1.3.1 Other Warehouses:**

  - > Dixon, IL

  - > Medina, OH

  - > Mesquite, NV

  - > Montgomery, NY

  - > Sikeston, MO

  - > Waco, TX

  - > Woodburn, OR

### **1.3.2 Shipping & ETA:**

  - > Typical ETA: \~7 days from order processing

  - > Orders ship to DK Hardware warehouse

## **1.4 OTHER WAREHOUSEs - DROPSHIP ELIGIBILITY RULES**

Dropship from other warehouses is allowed **only if ALL conditions are met:**:

  - > Small Parcel = Yes

  - > NOT Heavy / Freight / Clay

  - > NOT Canada / International Customer

**❌ If any condition fails → No Dropship (must process In-House)**

## **1.5 OTHER WAREHOUSE PO & NOTES RULES**

For Other Warehouse orders:

**1.5.1 PO Rules**

  - > Store PO only

  - > Do NOT include DKH- prefix

**1.5.2 Shipping:**

  - > Ship to DK Hardware warehouse

**1.5.3 Internal Notes Must Include:**

  - > Vendor: Do It Best

  - > Selected warehouse location

  - > ETA: 7 days

  - > Any Big / Freight restrictions

**SECTION 3 — DECISION MATRIX**

Table 1 - Do it Best (Out-of-Stock Orders and Other Warehouses) Decision Matrix

| **Scenario**                      | **Condition** | **Action**                       |
| --------------------------------- | ------------- | -------------------------------- |
| Item OFS in all warehouses        | TRUE          | Process as OFS order             |
| Item available in other warehouse | TRUE          | Process Other Warehouse In-House |
| Small parcel item                 | Eligible      | Dropship allowed                 |
| Heavy / Freight / Clay item       | Any           | In-House only                    |
| Canada / International CX         | Any           | In-House only                    |
| Item becomes available from OFS   | TRUE          | Move from OFS → Re-evaluate      |
