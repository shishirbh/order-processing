# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 Vendor Overview**

  - > Do it Best vendor orders are processed only by website

  - > Primary Warehouse is in Lexington, SC

  - > Order Processing: Daily

  - > Weekly Minimum Order Value (MOV) is $2,500

  - > Last Weekly Orders are placed on Sunday

  - > ETA (Main In-House) to DK Hardware Warehouse is Tuesday.

  - > Shipping Method is RSC Truck only

**1.2 Mandatory OFS Order Check**

## **Before processing ANY Main order:**

  - > Review OFS Vendor Orders

  - > Check if the item:

<!-- end list -->

  - > Is already sitting in an OFS order

  - > Is now back in stock

<!-- end list -->

  - > If item is available:

<!-- end list -->

  - > Move it from OFS order

  - > Process using Dropship → Main → Other logic

❌ Never process a new Main order without checking OFS first.

Hard Stops:

📌 Cancelled → Discontinued

📌 OFS everywhere → Keep in OFS order

📌 Heavy/Freight/Clay OFS everywhere → Discontinued

## **1.3 When to Use Main In-House**

**Process via Main (Lexington, SC) if:**

  - > Item ships via RSC Truck

  - > Item is Heavy / Freight / Clay

  - > Customer is Canada / International

  - > Dropship is not allowed

  - > Mixed customer order requires all items in-house

  - > Item released from OFS order

## **1.4 Dropship Eligibility Rules**

Dropship is always evaluated and is Allowed ONLY if ALL are true:

  - > Small Parcel

  - > NOT RSC Truck

  - > NOT Heavy / Freight / Clay

  - > NOT Canada / International Customer

  - > Available near customer address

**❌ If any condition fails → No Dropship**

**1.4.1 In-House vs Dropship Decision:**

Table 1

| **Condition**             | **Action**                         |
| ------------------------- | ---------------------------------- |
| Passes all dropship rules | Dropship                           |
| RSC Truck                 | In-House only                      |
| Heavy / Freight / Clay    | In-House                           |
| Canada CX                 | In-House                           |
| OFS released              | In-House or Dropship (re-evaluate) |

## **1.5 PO, Shipping & Status Rules**

  - > **PO Field:** Store PO only (no DKH- prefix)

  - > **Dropship Shipping:** FedEx Ground (DKH account)

  - > **Main (In-House) Shipping:** RSC Truck only

  - > **Status:** New → Submitted to Vendor

**SECTION 2 — PROCESSING ORDERS**

# **2.1 PROCESSING A VENDOR ORDER — IN-HOUSE**

**Steps:**

**2.1.1 Step 1 — Open Vendor Order**

  - > VO Status: New → Submitted to Vendor

  - > Use existing VO if available; else Create New VO

**2.1.2 Step 2 —Add Items to Vendor Cart**

  - > Open Do It Best portal.

  - > Add items exactly as VO:
    
      - > SKU, Qty, Specs
    
      - > Match VO totals

  - > Verify price and availability

**2.1.3 Step 3 — Shipping:**

  - > RSC Truck → DK Hardware warehouse

**2.1.4 Step 4 — PO Field:**

  - > Store PO only (no DKH- prefix)

**2.1.5 Step 5 — Checkout and Submit.**

**2.1.6 Step 6 — Copy Extended Cost and Confirmation \#**

  - > Paste Confirmation \# in VO notes

**2.1.7 Step 7 — Add ETA: Tuesday**

**2.1.8 Step 8 — Finalize VO**

  - > Save VO

  - > Status: Submitted

  - > CX order linked automatically

### **  
2.2 PROCESSING A VENDOR ORDER — DROPSHIP** 

**Steps:**

**2.2.1 Step 1 — Open Vendor Order**

  - > Open VO in portal

  - > Status: New → Submitted to Vendor.

**2.2.2 Step 2 — Verify Items**

  - > SKU, description, image, price

  - > Only include eligible items for dropship

**2.2.3 Step 3 — Shipping Info**

  - > Use DKH Shipping Account (FedEx / UPS)

  - > Select warehouse closest to customer

  - > Carrier: FedEx Group → FedEx Ground

**2.2.4 Step 4 — Checkout**

  - > Place order

  - > Status updates automatically

**2.2.5 Step 5 — Final Notes**

  - > Copy Extended Cost & Confirmation \# → VO notes

  - > ETA recorded if applicable (usually immediate for dropship)

# **2.3 PROCESSING A CUSTOMER ORDER — IN-HOUSE**

**Steps:**

**2.3.1 Step 1 — OFS Check**

  - > Review OFS Vendor Orders.  
    > If item was previously OFS and is now in stock → include in current order.

  - > If still OFS → keep in OFS queue.

**2.3.2 Step 2 — Process Incoming Order**

  - > Open Process Incoming Order in BO.

  - > Assign item(s) to Do It Best.

  - > Check if item is eligible for dropship (see dropship rules):
    
      - > If not eligible, proceed as Main In-House.

**2.3.3 Step 3 — Payment**

  - > Charge customer unless the order is:
    
      - > Amazon
    
      - > Zoro
    
      - > Net-30

  - > Status after payment: Coming In-House

  - > Click Process Changes → VO number generated

**2.3.4 Step 4 — Final CX Order Notes**

  - > Record:
    
      - > Item assigned to Main
    
      - > ETA Tuesday
    
      - > Extended Cost and VO number will be added after VO submission

# **2.4 PROCESSING A CUSTOMER ORDER — DROPSHIP**

**Steps:**

**2.4.1 Step 1 — OFS Check**

  - > Same as In-House: always check OFS first.

**2.4.2 Step 2 — Assign Item**

  - > Open Process Incoming Order in BO

  - > Assign item(s) to Do It Best

  - > Evaluate Dropship eligibility:
    
      - > Small Parcel
    
      - > NOT RSC Truck
    
      - > NOT Heavy / Freight / Clay
    
      - > Customer NOT Canada / International
    
      - > Warehouse near CX address

**2.4.3 Step 3 — Vendor Order**

  - > Create VO:
    
      - > Shipping = Dropship
    
      - > Inline Comment: 24 Hours Request

<!-- end list -->

  - > Click Process Changes → New VO number generated
