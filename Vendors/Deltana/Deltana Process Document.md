# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 Vendor Overview**

  - > Deltana orders are processed through Backoffice → Email.

  - > All communication must follow Deltana formatting rules.

  - > Elizabeth is Deltana Representative. We communicate only via email <span class="underline">elizabeth@deltana.net</span>

> → Elizabeth MUST be CC’d on every email sent to the vendor
> 
> → Check email requirements mentioned in Section 2

### **1.2 Minimum Order Requirements**

**1.2.1 In-House Orders**

  - > Minimum Order Value (MOV) is $75

  - > Process through Backoffice → Email

**1.2.2 Dropship Orders**

  - > Florida (FL) → Minimum Order Value (MOV) is $100

  - > Other U.S. States → Minimum Order Value (MOV) is $500

**1.2.3 Edge Cases**

  - > If an order is $495 and going outside FL → Do NOT send. Request add-on or convert to in-house if possible.

  - > If an order is just over the minimum (e.g., $502), still valid.

  - > If a customer wants expedited shipping, dropship is allowed only if minimums are met.

## **1.3 Email Requirements - MANDATORY**

Every Deltana order email MUST:

  - > CC: Elizabeth in ALL emails

  - > Use the correct email template:
    
      - > Standard
    
      - > Next Day
    
      - > Second Day

  - > Have no internal product notes.

  - > Have correct subject line, especially for expedited orders:
    
      - > NEXT DAY – Dropship Order
    
      - > SECOND DAY – Dropship Order

  - > Include correct customer shipping address for dropship.

**SECTION 3 — PROCESSING ORDERS**

# **3.1 INTERNAL PROCESSING STEPS (Important & Audit-Critical)**

### **3.1.1 Step 1 — Verify & Move Order to Submit**

### Before submitting, confirm:

  - ### Pricing correct

  - ### Quantities accurate

  - ### Shipping method selected

  - ### Meets minimum order value

  - ### No restricted notes

### Then → Move vendor order to SUBMIT.

### **3.1.2 Step 2 — Process Through Email**

  - ### In Backoffice:

  - ### Click By Email

  - ### Select correct shipping template

### **3.1.3 Step 3 — Insert Correct Draft Template**

### Based on shipping type:

  - ### Standard In-House  
    → Use In-House Standard draft.

  - ### Dropship – Standard Ground  
    → Use the Dropship Standard draft.

  - ### Dropship – NEXT DAY / SECOND DAY  
    → Must include BOTH:

<!-- end list -->

  - ### Correct subject line

  - ### Shipping instruction lines:

<!-- end list -->

  - ### SHIP CUSTOMER NEXT DAY

  - ### SHIP CUSTOMER SECOND DAY

### **3.1.4 Step 4 — Final Pre-Send Check**

### Before hitting send, confirm:

### ✔ Elizabeth is CC’d

### ✔ Minimum value is met

### ✔ Correct subject line

### ✔ Correct email draft inserted

### ✔ Customer address is correct (dropship only)

### ✔ No internal notes remain

**SECTION 4 — DECISION MATRIX**

Table 2 - Deltana Processing Decision Matrix

| **Scenario**             | **Condition**                | **Action**                                            |
| ------------------------ | ---------------------------- | ----------------------------------------------------- |
| In-House Order           | Total ≥ $75                  | Process through Backoffice → In-House template        |
| In-House Order           | Total \< $75                 | Ask customer to add items OR hold until threshold met |
| Dropship to FL           | Total ≥ $100                 | Process as Dropship                                   |
| Dropship to FL           | Total \< $100                | Ask for add-on OR convert to in-house                 |
| Dropship to Other States | Total ≥ $500                 | Process as Dropship                                   |
| Dropship to Other States | Total \< $500                | Ask for add-on OR convert to in-house                 |
| Dropship Standard        | Customer selected Standard   | Use Standard Ground template                          |
| Dropship Expedited       | Customer selected Next Day   | Use NEXT DAY template + add shipping line             |
| Dropship Expedited       | Customer selected Second Day | Use SECOND DAY template + add shipping line           |
| Any Order                | Elizabeth not CC’d           | STOP → Add CC                                         |
| Any Order                | Internal notes present       | STOP → Remove notes                                   |
| Any Dropship             | Wrong or incomplete address  | STOP → Correct the address                            |
