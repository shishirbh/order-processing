# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 Vendor Overview**

  - > Order processing method is through Vendor Website.

  - > Minimum Order Value (MOV) for Vendor Orders: $750

### **Customer Orders:**

  - > If customer order value is below $750, move items to in-house order.

  - > Can attempt dropship only when customer order value is ≥ $750.

### **Backorders:**

  - > Out-of-stock with no ETA → still allowed.

# **ALWAYS AVOID:**

> ❌ Dropshipping orders under $750  
> ❌ Skipping shipping cost checks for expedited orders

### **1.2 Dropship Rules & Restrictions**

**1.2.1 Dropship Is Allowed ONLY When:**

  - > Customer order value ≥ $750.

  - > Shipping cost is reasonable.

  - > Shipping method is one of the following:
    
      - > Standard Ground
    
      - > 1-Business Day
    
      - > 2-Business Day

### **1.2.2 Dropship Restrictions:**

  - > If order value \< $750 → must be handled in-house.

  - > For expedited shipments (1-Day / 2-Day):
    
      - > Must first check shipping cost on the website.
    
      - > Dropship only if shipping cost is reasonable.

**1.4 Backorder Rules & ETA Handling**

#### **Acceptable Backorder Scenarios**

  - > DK Hardware ETA shows 1–2 weeks → Backorder allowed

  - > Vendor shows out-of-stock with NO ETA → Backorder allowed

**SECTION 2 — PROCESSING ORDERS**

# **2.1 PROCESSING A VENDOR ORDER — IN-HOUSE**

### **When to Use In-House:**

  - > Customer order value \< $750

  - > Dropship shipping cost is unreasonable

  - > Risk mitigation or internal inventory preference

### **STEPS:**

### **2.1.1 Step 1 — Verify Items**

  - > Confirm correct SKU

  - > Verify stock status

  - > Confirm customer approval if backordered

### **2.1.2 Step 2 — Add Items to Cart**

  - > Add items on vendor website

### **2.1.3 Step 3 — PO Entry**

  - > Enter PO correctly

  - > Ensure total ≥ $750 (vendor order level)

### **2.1.4 Step 4 — Checkout**

  - > Enter DK Hardware shipping information

  - > Double-check shipping method

### **2.1.5 Step 5 — Verification**

  - > Stock accuracy

  - > Shipping accuracy

  - > Shipping address

### **2.1.6 Step 6 — Place Order**

  - > Submit order

  - > Save confirmation number

  - > Update internal system

# **2.2 PROCESSING A CUSTOMER ORDER — DROPSHIP**

### **Dropship Eligibility Checklist:**

  - > Customer order value **≥ $750**

  - > Shipping cost verified and reasonable

  - > Shipping method supported

**STEPS:**

**2.2.1 Step 1 — Review Order**

  - > Identify shipping type (Standard / 1-Day / 2-Day)

  - > Confirm order value ≥ $750

**2.2.2 Step 2 — Enter Shipping Details**

  - > Customer full address

  - > Customer email

  - > PO must match internal system

**2.2.3 Step 3 — Final Verification**

  - > Correct items

  - > Correct shipping method

  - > Acceptable shipping cost

  - > Correct customer information

**2.2.4 Step 4 — Place Order**

  - > Submit order via vendor website

  - > Copy confirmation number

  - > Update internal status:  
    > **New → Dropship → Submitted**
