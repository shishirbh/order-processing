# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 MANDATORY FIRST STEP - Orders Classification**

Before taking any action, classify the order across four dimensions:

  - > **Destination:**
    
      - > US (Mainland)
    
      - > US Islands (HI, AK, PR)
    
      - > International

  - > **Tax Status:**
    
      - > Tax-Exempt State
    
      - > Non-Tax-Exempt State

  - > **Shipping Type and Minimum Order Value:**
    
      - > Minimum order requirements vary based on item size and shipping type as discussed in Table 1.

Table 1 - Minimum Requirements for Prepaid Freight

| **Shipment Type**            | **Prepaid Minimum Order Value (MOV)** |
| ---------------------------- | ------------------------------------- |
| Shipments 96” and longer     | $3,000.00 or more                     |
| Shipments less than 96”      | $2,000.00 or more                     |
| 48” skids or FedEx shipments | $1,000.00 or more                     |

⚠️ **No processing may begin until classification is complete.**

⚠️ **Use Fedex Account if not reaching minimum on the order for inhouse orders too**

### **1.2 Vendor Rules**

  - > EPCO vendor orders are processed **only by email** using Backoffice (BO).

  - > Orders can be submitted every Friday or whenever they reach the minimum requirements

  - > Dropship is allowed ONLY where explicitly permitted (Section 1.4)

  - > Freight orders require verification before submission

  - > FedEx and BOL instructions must NEVER be mixed

  - > Monitor inbox after order submission for:
    
      - > Confirmations
    
      - > Reverts
    
      - > Stock updates
    
      - > Dimension requests

  - > Respond promptly to avoid delays

## **ALWAYS AVOID:**

❌ Dropshipping non-tax-exempt states  
❌ Dropshipping international freight  
❌ Mixing FedEx + BOL instructions  
❌ Skipping freight verification

### **1.3 IN-HOUSE ORDERS**

Process as **In-House** if **any** of the following apply:

  - > Order is International

  - > Order ships to a Non-Tax-Exempt State

  - > Order is not approved for dropship

  - > Order is shipping to HI, AK, or PR

  - > Order is International Freight

### **1.4 DROPSHIP ORDERS**

### **When to Dropship:**

  - ### Dropship items longer than 48" directly to the customer.

  - ### Vendors often struggle to meet minimum requirements; if the order does not meet minimum, ship using our FedEx account.

### **Eligibility Criteria (ALL must be true):**

An order qualifies for Standard Dropship only if:

  - > Shipping destination is within the United States

  - > Shipping destination is a Tax-Exempt State

  - > Order is NOT international

  - > Order is NOT shipping to HI, AK, or PR

If any condition fails → Dropship is NOT allowed.

**Always Verify:  
**✔ Customer address accuracy  
✔ Tax-exempt status  
✔ Correct email template usage

**1.5 CUTTING ITEMS**

  - > EPCO can provide cutting services for long items upon customer request.

  - > If a customer requires an item to be cut:

  - > EPCO will complete the cutting and process the order as a normal dropship.

  - > A cutting fee is typically added to the dropship invoice.

**SECTION 3 — PROCESSING ORDERS**

# **3.1 PROCESSING A VENDOR ORDER — IN-HOUSE**

### **3.1.1 Step 1 — Verify & Move Order to Submit**

### Before submitting, confirm:

  - ### Pricing correct

  - ### Quantities accurate

  - ### Shipping method selected

  - ### Meets minimum order value

  - ### No restricted notes

  - > All small items are added

### Then → Move vendor order to SUBMIT.

### **3.1.2 Step 2 — Process Through Email**

  - ### In Backoffice:

  - ### Click By Email

  - ### Select correct shipping template

### **3.1.3 Step 3 — Insert Correct Draft Template**

### Based on shipping type:

  - ### Standard In-House  
    → In-House Order Template (Warehouse / International orders)

### **3.1.4 Step 4 — Final Pre-Send Check**

### Before hitting send, confirm:

### ✔ Correct subject line

### ✔ Correct email draft inserted

### ✔ No internal notes remain

# **3.2 PROCESSING A CUSTOMER ORDER — STANDARD DROPSHIP**

3.2.1 Step 1 — Create the vendor order in the BO first  
3.2.2 Step 2 — Verify:

  - > Customer shipping address (mandatory)

  - ### Pricing correct

  - ### Quantities accurate

  - ### Shipping method selected

  - ### Meets minimum order value

  - ### No restricted notes

  - > Include Cutting Info below the item Description (if required)

3.2.3 Step 3 — Submit the request using:

  - > DK Hardware Back Office email

  - > Standard Dropship Email Template

3.2.4 Step 4 — Always include:

  - > Our FedEx account \#632647611

# **3.3 PROCESSING A CUSTOMER ORDER — FREIGHT DROPSHIP**

3.3.1 Step 1 — Request shipping dimensions from EPCO  
3.3.2 Step 2 — Share dimensions with Marina  
3.3.3 Step 3 — Confirm if shipping is acceptable  
3.3.4 Step 4 — Once approved:

  - > Proceed with dropship

  - > Use Bill of Lading (BOL)

3.3.5 Step 5 — Submit using Freight Dropship Email Template

**SECTION 4 — DECISION MATRIX**

Table 2 - EPOC Processing Decision Matrix

| **Scenario**                                    | **Action**                   | **Notes**                                 |
| ----------------------------------------------- | ---------------------------- | ----------------------------------------- |
| Order ships to International destination        | In-House only                | Dropship not allowed.                     |
| Shipping to HI / AK / PR                        | In-House only                | Use FedEx account if MOV not met.         |
| Shipping to Non-Tax-Exempt State                | In-House only                | Dropship not permitted.                   |
| Shipping to US Mainland + Tax-Exempt State      | Dropship allowed             | All eligibility conditions must be met.   |
| Item longer than 48″                            | Prefer Dropship              | Ship directly to customer.                |
| Order does not meet minimum freight requirement | Use FedEx account            | Applies to In-House orders as well.       |
| Freight shipment                                | Verify shipping first        | Request dimensions → confirm with Marina. |
| Freight dropship approved                       | Use BOL                      | Do not mix FedEx + BOL instructions.      |
| Customer requires cutting service               | Include cutting instructions | Fee added to dropship invoice.            |
| Order meets MOV for prepaid freight             | Process normally             | Refer to Table 1 for thresholds.          |
| Shipment 96″ or longer                          | MOV $3,000                   | Required for prepaid freight.             |
| Shipment less than 96″                          | MOV $2,000                   | Required for prepaid freight.             |
| 48″ skid / FedEx shipment                       | MOV $1,000                   | Required for prepaid freight.             |
