# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 Vendor Overview**

  - > Primary ordering method is via Vendor Website. Email is used only in specific low-value expedited cases.

  - > Dropship is allowed only if order value \> $350 (with special cases in Section 1.3).

  - > Backorders are allowed for all items. Most items have an eta of 2-3 weeks if they are on backorder. Always check the ETA by clicking on ‘available’ or by using the chat option on the vendor portal.

  - > Always select or request Blind Shipping for all dropship orders

### **ALWAYS AVOID:**

> ❌ Processing expedited orders \< $50 on website  
> ❌ Ignoring fragile warnings  
> ❌ Dropshipping without blind shipping  
> ❌ Processing build-to-configure without ETA  
> ❌ Guessing stock on large OOS items

**1.2 Order Types & Processing Methods**

Table 1

| **Order Type**                         | **Method**        | **Notes**                                                 |
| -------------------------------------- | ----------------- | --------------------------------------------------------- |
| In-house - Standard Vendor Order       | Website           | Default method for all non-expedited, non-dropship orders |
| Dropship ≥ $350                        | Website           | Use FedEx account                                         |
| Dropship 1-Day / 2-Day Shipping \< $50 | Email and Website | Either Method can be used.                                |

### **1.3 Dropship Eligibility Rules**

Dropship is allowed ONLY IF AT LEAST ONE of the following is true:

  - > Order value \> $350

  - > Item is Build-to-Configure

  - > Item is Large + Out of Stock (requires approval)

  - > Customer selected 1-Day or 2-Day Shipping

🚫 Never dropship below $350 unless one of the above exceptions applies.

### **1.4 Fragile Item Handling (MANDATORY)**

If the vendor site displays:

> *“Fragile: Due to the fragile nature of this item… additional charges apply”*

You MUST:

1.  > Calculate additional shipping charge (based on quantity)

2.  > Update customer order with added cost

3.  > Move order to **Problem Orders → Customer Service**

4.  > Add internal notes clearly stating reason

🚫 Never ignore fragile warnings

### **1.5 Large Out-of-Stock Items**

Rules:

  - > Dropship only if Seclock confirms availability

  - > Must verify directly with Seclock Customer Service

  - > No assumption allowed

Example: 178SA 96

### **1.6 Build-to-Configure Items**

Rules:

  - > Dropship allowed only if order \> $350

  - > ETA must be confirmed with Seclock CS

  - > Do not process without ETA confirmation

Example text:

> “*We build to configure the exact specs for 45H0LB14R626VIT…*”

### **1.8 Email Processing Rules (Orders \< $50)**

### When Email Is Required

  - > Order value \< $50

  - > Shipping method = 1-Day or 2-Day

🚫 Website processing is NOT allowed in this case

**SECTION 2 — PROCESSING ORDERS**

# **2.1 PROCESSING VENDOR ORDER — IN-HOUSE**

## **2.1.1 Verify Items**

  - > Confirm item validity

  - > No restricted/discontinued items

  - > Identify dropship eligibility

## **2.1.2 Step 2 — Add Items to Cart**

  - > Exact item match

  - > PO number must match internal record

  - > Prices must match website (no deviation)

## **2.1.3 Step 3 — Check Eligibility**

  - > Free shipping

  - > Dropship qualification

  - > Expedited rules

### **2.1.4 Step 4 — Internal Notes**

  - > Record cart total

  - > Note fragile warnings clearly

**2.1.5 Step 5 — Stock Check**

  - > Confirm availability

  - > Discontinued → **STOP**

  - > Large OOS → Confirm with Seclock CS

**2.1.6 Step 6 — Checkout**

  - > Select DK Hardware address

  - > Apply **Blind Shipment**

**2.1.7 Step 7 — Review**

  - > Items

  - > Shipping address

  - > PO number

  - > Notes

  - > Screenshot shipping page → Attach to VO

**2.1.8 Step 8 — Place Order**

  - > Submit order

  - > Copy confirmation number

  - > Paste into Backoffice vendor order

# **2.2 PROCESSING CUSTOMER ORDER — DROPSHIP**

### **Dropship When ANY Apply:**

  - > Order ≥ $350

  - > 1-Day / 2-Day shipping selected

  - > Vendor allows dropship

  - > Large OOS item confirmed by Seclock

  - > Build-to-configure item (ETA confirmed)

## **Steps:**

**2.2.1 Step 1 —** **Review Order**

  - > Identify shipping type

  - > Check value threshold

  - > If \< $50 + expedited → Email only

**2.2.2 Step 2 — Shipping Details**

  - > Full customer address

  - > Correct PO number

  - > Blind shipping applied

**2.2.3 Step 3 — Final Checks**

  - > Status: New → Dropship

  - > Correct shipping method

  - > FedEx account used (website)

**2.2.4 Step 4 — System Update**

  - > Confirmation number

  - > Dropship flag

  - > Shipping method

  - > Fragile charges (if any) - Some lengthy items on seclock have a handling fee that is displayed on the item page
