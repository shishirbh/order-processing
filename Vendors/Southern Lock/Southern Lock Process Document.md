# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 Key Rules & Notes**

  - > Minimum Order Requirement is $25.

  - > Vendor allows in-house as well as dropship.

  - > Backorder is allowed.

  - > ETA is 5 days if item is in-stock

**Do’s ✅**

  - > Always verify product availability on the Southern Lock vendor portal before placing the order.

  - > Ensure vendor order value meets the $25 minimum requirement.

  - > Merge duplicate items before submitting vendor orders.

  - > Record confirmation numbers and order totals in Vendor Order notes.

  - > Verify that the vendor portal price matches the Vendor Order price.

**Don’ts ❌**

  - > Do not process orders below the $25 minimum value.

  - > Do not dropship items that are out of stock on the vendor portal.

  - > Do not place vendor orders without verifying product details.

**SECTION 3 — DECISION MATRIX**

Table 2 - Southern Lock Processing Decision Matrix

| **Scenario**             | **Action**             | **Notes**                      |
| ------------------------ | ---------------------- | ------------------------------ |
| Order value \< $25       | Hold order             | Must meet vendor minimum       |
| Order value ≥ $25        | Process order          | In-House or Dropship allowed   |
| Item in stock            | Process order          | ETA approx. 5 days             |
| Item backordered         | Allow backorder        | Update ETA when available      |
| Item discontinued        | Verify replacement     | Compare with portal listing    |
| Dropship order           | Use vendor website     | Customer address required      |
| In-house order           | Submit via email       | Use Back Office email option   |
| Expedited order required | Select faster shipping | Choose 1–2 business day option |
