**SECTION 2 — PROCESSING ORDERS**

# **2.1 PROCESSING A VENDOR ORDER — IN-HOUSE**

### **2.1.1 Step 1 — Cross-Check Customer (CX) Order**

  - > Match each item in CX order to the correct Strybuc SKU.

  - > Verify all items assigned correctly.

  - > If SKU missing → Check Previous Invoices/POs if not found Ask Elina / Kiril / Alex.

### **2.1.2 Step 2 — Verify Price**

  - > Check price on Strybuc website.

  - > If shows “Call for price”:
    
      - > Check prior invoice/previous PO.
    
      - > If first-time → ask Alex.

### **2.1.3 Step 3 — Add Strybuc Item Code (Mandatory)**

  - > Copy item code below line item.

  - > Add inside item notes.

### **2.1.4 Step 4 — Check for Substitution**

  - > If item was previously substituted with CRL / FHC / US Lock:
    
      - > Search and assign correct Strybuc code.
    
      - > Update notes.

### **2.1.5 Step 5 — MOQ (Minimum Order Qty)**

*Rules:*

  - > If MOQ \> ordered qty → increase to MOQ.

  - > If sold as multiple/set → adjust to nearest multiple.

*Examples:*

  - > MOQ = 120; CX Ordered = 60 → Change to 120.

  - > MOQ = 2; CX Ordered = 1 → Change to 2.

  - > Sold in multiples of 20; CX ordered 30 → Change to 20 or 40 (choose logically based on order purpose—ask seniors if unclear).

### **2.1.6 Step 6 — Final Processing**

  - > Submit vendor order by email using vendor draft.

  - > After sending:  
    > Monitor Order Acknowledgement (OA).
    
      - > Verify every price & quantity.
    
      - > If discrepancy:
        
          - > Email Alex for corrected OA immediately.

# **2.2 PROCESSING A VENDOR ORDER — DROPSHIP**

### **2.2.1 Customer Order (CX) Processing**

  - > Go to Process Incoming Order.

  - > Assign items to Strybuc Vendor.

  - > Go to:  
    > Strybuc Warehouse → Create New Order → Dropship.  
    > Status changes:  
    > New Orders → Placed for Dropship.

  - > Click Process Changes → System generates vendor order number.

### **2.2.2 Vendor Order (Submission)**

  - > Check all in-house rules:
    
      - > Pricing
    
      - > MOQ
    
      - > Multiples
    
      - > Strybuc code

  - > Click By Email → Attach draft → Send.

  - > Monitor reply in a few hours.
