# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 Vendor Overview**

  - > Primary Email: *<span class="underline">info@ariavetri.com</span>*

  - > Order Submission Days: Mostly Fridays unless urgent

  - > Shipping Method: FedEx (always DK FedEx account) - Need to ask Elina (UPS)

  - > Preferred Order Method: Dropship for ALL orders except long items and extrusions

  - > Special Allowance: CRL substitution allowed when original item is unavailable or when long items require alternative sourcing.

## **1.2 INTERNAL WORKFLOW NOTES**

  - > Default submission day: Friday

  - > Standard dropships can be done any day if urgent

  - > Every long-item dropship → label needed

  - > Always check dimensions — missing = notify Kirill

  - > CRL substitutions allowed with approval

  - > Keep email communication clean and audit-ready

## **1.3 Shipping Rules**

### **1.3.1 Default Rule — Dropship All Orders**

  - > All standard items (≤ 95 inches) must be dropshipped directly from the vendor.

  - > Vendor ships using our FedEx account, not the website’s estimated shipping cost.

### **1.3.2 Items Over 95 Inches (Long Items)**

Vendor cannot ship long items without our help. For all items \> 95 inches:

  - > Generate FedEx shipping label

  - > Generate Pickup ID

  - > Email both to vendor at [**<span class="underline">info@ariavetri.com</span>**](mailto:info@ariavetri.com)

  - > Vendor will then ship using our FedEx account.

**Notes / Traps to avoid:**

  - > Never submit long items without a DK-generated label — vendor will hold the order.

  - > Always verify dimensions before categorizing long vs standard.

### **1.3.3 Shipping Cost Notes**

  - > Website shipping cost = approximation only

  - > Actual cost using our FedEx account is almost always lower

  - > Exception: Very long items may have high dimensional charges

## **1.4 Missing Weight/Dimension Handling**

If an item lacks weight or dimensions in Back Office:

  - > Share the item details with Kirill

  - > Kirill updates the system with correct weight/dimensions

  - > Reprocess shipping calculation if needed

**Why this matters:  
**Accurate dimensions are required for correct label generation and avoiding unexpected FedEx fees.

**Example:  
**Item length shows blank → treat as potentially long → confirm with Kirill before categorizing.

**SECTION 2 — PROCESSING ORDERS**

# **2.1 STANDARD DROPSHIP ORDERS (Items ≤ 95 Inches)**

**When to Use:**

  - > All normal items

  - > Website shows Business Day shipping normally

**Process:**

  - > Process order through Aria Vetri website

  - > Ignore website shipping cost (vendor uses our FedEx account)

  - > Submit the order normally

  - > No label required

  - > Track completion

**Example:  
** DKH-2245111 → 72-inch hardware → Standard dropship → Submit order → Done.

### **2.2 DROPSHIP ORDERS FOR LONG ITEMS (\> 95 Inches)**

**When to Use:**

  - > Any line item longer than 95 inches

**Process:**

  - > Process order on Aria Vetri website

  - > After submitting the order, send Marina:
    
      - > Ship-from address
    
      - > Ship-to customer address
    
      - > Order number

  - > Marina generates:
    
      - > FedEx shipping label
    
      - > Pickup ID

  - > Marina attaches the label to the vendor order

  - > Email label + pickup ID to **info@ariavetri.com**

**Example:  
**DKH-2299001 → 102-inch extrusion → Dropship but label required → Follow long-item steps.

### **2.3 IN-HOUSE ORDERS (Extrusions Only)**

**When to Use:**

  - > When ordering extrusion packs of 10–20

  - > Vendor charges packing fees for extrusions; in-house consolidation reduces cost

**Process:**

  - > Process in-house order through vendor website

  - > Send ship-from + ship-to details to Marina

  - > Marina creates FedEx label + Pickup ID

  - > Email label to vendor

  - > Vendor ships using DK account

**Notes:**

  - > Treat extrusions separately from long items

  - > Similar process but marked as in-house pack ordering

**Example:  
**Order 14 pcs of aluminum extrusion → Mark as in-house → Follow extrusion workflow.
