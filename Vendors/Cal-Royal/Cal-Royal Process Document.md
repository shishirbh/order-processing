# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 Key Rules & Notes**

  - > Cal-Royal vendor orders are processed via Vendor Website only.

  - > In-house orders under $50.00 are subject to a service charge. (Updated: 2026-04-26)

  - > Orders cannot be submitted online if order total is less than $100.00. (Updated: 2026-04-26)

  - > Dropship may be attempted for order value above $100.

  - > Always verify stock before processing

  - > Dropship eligibility depends on order value & shipping method as discussed in section 1.2.

  - > All expedited orders MUST use DK Hardware FEDEX account \#632647611

  - > Customer contact details must be entered correctly for all dropship orders

  - > 30-inch plates ship by freight — ALWAYS escalate to Elina

#### **Always Avoid:**

❌ If customer order ≤ $100 → NO dropship  
❌ If FEDEX missing → DO NOT SUBMIT  
❌ Never include internal notes

**1.2 Shipping & Processing Rules**

#### **1.2.1 Vendor Orders – In-House**

**When This Applies:**

  - > DK Hardware is ordering stock for internal inventory.

**Processing Method:**

  - > Website ONLY

**Rules:**

  - > If item is **in stock** → proceed normally

  - > If item is out of stock →  
    > → STOP  
    > → Check ETA with Cal-Royal before placing order

**Example:**

  - > Vendor order shows “Out of Stock” on website  
    > → Email/confirm ETA  
    > → Proceed only after ETA confirmation

**Always Avoid:**

❌ Placing vendor orders without ETA confirmation  
❌ Assuming backorder timelines

#### **1.2.2 Customer Orders – Standard Ground**

**Processing Method:**

  - > Website ONLY

**Dropship Eligibility:**

  - > Order value **above $100** → Dropship **may be attempted**

  - > Order value **≤ $100** → Dropship **NOT allowed** unless approved

**Rules:**

  - > Always verify live stock on Cal-Royal website

  - > Confirm dropship eligibility before checkout

  - > Enter full customer details correctly

**Example:**

  - > Customer order = $145, Standard Ground  
    > → Stock available  
    > → Dropship via website  
    > → Enter customer shipping details

**Always Avoid:**

❌ Dropshipping orders below $100  
❌ Processing without stock verification

#### **1.2.3 Customer Orders – Expedited (1 or 2 Business Day)**

**Processing Method:**

  - > Website ONLY

**Mandatory FEDEX Rule (Critical):**

  - > ALL expedited orders MUST ship using DK Hardware FEDEX account

  - > Account Number: \#632647611

**Required Website Action:**

  - > Add instruction in **comment box next to “Best Way”**:  
    >   
    > *“Ship customer Next Day / 2nd Day COLLECT using DK Hardware FEDEX Account \#632647611.”*

**Rules:**

  - > FEDEX account is mandatory

  - > No vendor account allowed

  - > Customer phone number must be present

**Example:**

  - > Customer selects Next Day Air  
    > → Website order  
    > → Add FEDEX account instruction  
    > → Verify customer phone & address

**Always Avoid:**

❌ Forgetting FEDEX account instructions  
❌ Allowing vendor-paid expedited shipping  
❌ Leaving comment box blank

#### **1.2.4 Special Freight Rule – 30-Inch Plates**

**Critical Exception Rule:**

  - > ALL 30-inch plates ship by FREIGHT

**Action Required:**

  - > STOP processing

  - > Escalate to Elina immediately

**Never Self-Decide:**

  - > Freight routing

  - > Cost handling

  - > Dropship eligibility

**Always Avoid:**

❌ Processing 30-inch plates without escalation  
❌ Treating as standard ground

**SECTION 2 — PROCESSING ORDERS**

# **2.1 INTERNAL PROCESSING STEPS (for In-House and Dropship Orders)**

**2.1.1 Step 1 — Review Order**

  - > Verify correct SKUs and quantities

  - > Confirm live stock on Cal-Royal website

  - > Check order value for dropship eligibility

  - > Identify expedited shipping

  - > Flag 30-inch plates immediately

**2.1.2 Step 2 —** **Determine Processing Method**

Table 1

| **Scenario**               | **Method**                 |
| -------------------------- | -------------------------- |
| In-stock / Standard orders | Website                    |
| Vendor order OOS           | Website after ETA          |
| Special cases              | Email (only if instructed) |

**2.1.3 Step 3 —** **Enter Customer Information**

**For Dropship Orders:**

  - > Customer name

  - > Full address

  - > Phone number (mandatory)

**For Expedited Orders:**

  - > Add FEDEX account instructions in comment box

**2.1.4 Step 4 — Final Pre-Submit Checks**

  - > Shipping method correct

  - > FEDEX account added (if expedited)

  - > Stock or ETA confirmed

  - > Dropship allowed by order value

  - > No internal notes visible to vendor

**2.1.5 Step 5 — Submit only after ALL checks pass**
