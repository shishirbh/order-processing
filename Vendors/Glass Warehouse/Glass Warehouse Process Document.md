# **SECTION 1 — OVERVIEW & DEFINITIONS**

### **1.1 General Rules**

  - > ETA must be checked for every Glass Warehouse order

  - > Glass Warehouse dropships are NOT processed directly by DK

  - > Every GW dropship order MUST be added to the Marina Processing Sheet

  - > If the same item was recently processed, stock does not need to be rechecked

  - > If any approval is required, do NOT proceed without Marina confirmation

**Always Remember:  
Never** submit dropship order — **Marina does**

## **Always Avoid:**

❌ Processing a dropship directly  
❌ Skipping ETA verification  
❌ Assuming FedEx without dimension review  
❌ Ignoring width \>38" rule  
❌ Failing to add order to Marina sheet  
❌ Proceeding without Marina approval

⚠️ **<span class="underline">IMPORTANT NOTE:</span>**

Vendor orders for **Glass Warehouse** are not processed through the usual vendor ordering process. We only **create the vendor order and log the order details in the tracking sheet**, as shown in the example and figures below. The sheet entry serves as the processing record.

### **1.3 New / Never-Sold Product Handling**

→ If a product has **never been ordered before**,

  - > check the **same product in a different color** on the vendor website.

→ This helps verify:

  - > Packaging format

  - > Number of boxes

  - > Typical shipping method (FedEx vs freight)

→ Doing this helps **avoid FedEx shipping issues and unexpected freight costs**, especially when ordering larger quantities.

## **1.7 ETA Handling & Hold Orders**

### **→ If Item Has an ETA**

  - > Confirm customer acceptance

  - > If customer agrees to wait:
    
      - > Create the dropship record
    
      - > Place order on **Hold**
    
      - > Add to Marina sheet
    
      - > Clearly note:
        
          - > ETA
        
          - > “Customer approved wait”

🎯 **Why this matters:  
**Prevents accidental processing of delayed orders and ensures Marina is fully informed.

**SECTION 2 — DECISION MATRIX**

Table 3 - Glass Warehouse Processing Decision Matrix

| **Scenario**                  | **Quantity** | **Width** | **Item History** | **Action**                          |
| ----------------------------- | ------------ | --------- | ---------------- | ----------------------------------- |
| Same item recently processed  | Any          | Any       | Known            | Skip stock check, proceed           |
| New item, never sold          | Any          | Any       | Unknown          | Check same item in other color      |
| 1 unit                        | ≤38"         | Any       | Any              | FedEx likely acceptable             |
| 2 units                       | ≤38"         | Any       | Any              | FedEx likely acceptable             |
| 3+ units                      | Any          | Any       | Any              | Request dimensions & review freight |
| Any item                      | \>38"        | Any       | Any              | Freight review required             |
| Multiple items                | Any          | Any       | Any              | Combine dimensions, send to Marina  |
| Freight expensive             | Any          | Any       | Any              | Compare FedEx vs Freight            |
| ETA present & customer agrees | Any          | Any       | Any              | Hold order + note ETA               |
| No Marina approval            | Any          | Any       | Any              | STOP – do not proceed               |
