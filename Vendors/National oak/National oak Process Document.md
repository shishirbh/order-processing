### **1.1 Key Rules & Notes**

  - > Always Remember: Each warehouse is treated as a separate order with its own minimum and shipping rules.

**1.3 Free Shipping Rules**

  - > **Standard Rule:**
    
      - > Each warehouse must reach $375+ individually for free shipping.

  - > **If warehouse total \< $375:**
    
      - > Shipping is charged based on **item weight/size**.

  - > **Special Exception (ORL – Orlando):**
    
      - > **Always qualifies for free shipping**, even if order value is **below $375**
    
      - > Reason: *Prepaid Alternate Location Choice*

⚠️ **Do NOT apply ORL exception to any other warehouse.**

### **1.4 Identifying Freight Items**

  - > Freight items are marked with a **large truck icon**.

  - > There is also a small truck icon, indicting extra shipping even if the warehouse is above min order items with small truck icon do not qualify for free shipping from other warehouses, if item is not available in Miami then item should be dropshipped if customer paid enough shipping

**Freight Handling Rules:**

  - > If freight item is available in **Miami (Main Warehouse)** →  
    > ➜ Ships free

  - > If **not available in Miami** →  
    > ➜ Check shipping cost with Marina  
    > ➜ Confirm cost with customer  
    > ➜ Proceed as dropship only after approval

🚫 **Never assume free freight outside Miami.**

### **1.5 Dropship Logic (Critical)**

Dropshipping is **conditional**, not automatic.

**Scenario A: Item available in multiple warehouses, but NOT in one single warehouse**

  - > If customer **paid sufficient shipping** →  
    > ➜ Dropship from each warehouse

  - > If customer **did NOT pay sufficient shipping** →  
    > ➜ Check shipping with Marina  
    > ➜ Request customer approval **before processing**

⚠️ **Never split shipments without customer consent when extra shipping applies.**

### **1.6 ETA**

  - > Miami (Main Warehouse): \~ 2 days

  - > All Other Warehouses: 7–10 days

🚫 **Never promise faster ETA without written vendor confirmation.**

**SECTION 2 — PROCESSING ORDERS**

# **2.1 INTERNAL PROCESSING STEPS (for In-House and Dropship Orders)**

**2.1.1 Step 1 — Adding Items to Cart (Validation Step)**

Before adding any item, always verify:

  - > Item code

  - > Description

  - > Price

**2.1.2 Step 2 — Status Checks**

  - > If item shows **“Obsolete”** →  
    > ➜ Item is **Discontinued (DISC)** → **Do NOT order**

  - > If item note says **“All sales are final / Non-refundable”** →  
    > ➜ Item is being discontinued →  
    > ➜ **No returns allowed** → must inform customer **before processing**

📌 **Example:  
** If customer later requests a return on a “final sale” item → reject return immediately.

**2.1.3 Step 3 —** **In case of** **Freight Items**

  - > Freight items = truck icon.

  - > Miami stock → free shipping.

  - > Not in Miami → get freight cost from Marina + customer approval.

**2.1.4 Step 4 — Warehouse Assignment Review**

After all items are added:

  - > System automatically assigns items to available warehouses.

  - > Review each warehouse separately:

<!-- end list -->

  - > If warehouse total ≥ $375 →  
    > ➜ Process **In-House** (Free Shipping)

  - > If warehouse total **\< $375** →  
    > ➜ Shipping charged  
    > ➜ Evaluate dropship via FedEx account

**2.1.5 Step 5 — Apply Dropship Logic**

  - > Dropship only when allowed.

  - > If items are split across warehouses:
    
      - > Customer paid enough → dropship.
    
      - > Customer didn’t pay → confirm cost with Marina + get approval.

  - > Never split shipments without consent.

### **2.1.6 Step 6** **— Out-of-Stock (OOS) Management**

  - > If item is **not available in ANY warehouse**:
    
      - > Move item to Out-of-Stock Warehouse Profile
    
      - > Process separately at the end
    
      - > Do not hold other items

📌 **Best Practice:  
** Offer alternate vendors immediately to reduce order delay.

**2.1.7 Step 7** **— Final Order Placement Steps (Strict Sequence)**

  - > Remove all dropship items
    
      - > Process them individually first**  
        > **

  - > Return to main cart

  - > Confirm:
    
      - > Cart value
    
      - > Warehouse assignment
    
      - > Final shipping charges

  - > If everything matches → Click Process

  - > Enter:
    
      - > **Ordered By:** Your name
    
      - > **PO Number:** Customer PO

  - > Click Checkout

**2.1.8 Step 8** **— Order Confirmation**

  - > Copy order confirmation number

  - > Paste it into the vendor order record

  - > Ensure confirmation is logged same day

Note -

If warehouse total less than $375 see if any items can be moved from Miami or other warehouses to the warehouse to help reach $375.00

**SECTION 3 — DECISION MATRIX**

Table 1 - National Oak Processing Decision Matrix

| **Scenario**                | **Condition**       | **Action**                       |
| --------------------------- | ------------------- | -------------------------------- |
| Item shows “Obsolete”       | Status = Obsolete   | Do NOT order (DISC)              |
| Item marked “Final Sale”    | Non-refundable note | Inform customer; no returns      |
| Warehouse total ≥ $375      | Any warehouse       | In-House, Free Shipping          |
| Warehouse total \< $375     | Non-ORL             | Shipping charged                 |
| ORL warehouse               | Any value           | Free shipping                    |
| Freight item in Miami       | Available           | Free shipping                    |
| Freight item not in Miami   | Not available       | Check Marina → customer approval |
| Item in multiple warehouses | No single warehouse | Dropship only if shipping paid   |
| Shipping insufficient       | Any split order     | Get customer approval            |
| Item unavailable everywhere | All warehouses OOS  | Move to OOS profile              |
| Dropship items present      | Mixed cart          | Remove & process separately      |
| Order day not Friday        | Any                 | HOLD — do not submit             |
