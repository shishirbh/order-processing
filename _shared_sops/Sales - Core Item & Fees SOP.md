## **1. What Is a Core Fee?** 

## A core fee is a refundable deposit charged in addition to the product price.  
It exists to ensure the customer returns the used or old part (core) after receiving the new item.

**No return = No refund.  
Return confirmed = Full refund of the core fee with taxes applied on it.**

## **2. Core Fee Rules (Non-Negotiable)**

These rules are absolute. No exceptions.

  - > Core fee is **mandatory** NOT optional

  - > Core fee is **refundable**, but **never automatic**

  - > Core fee **cannot be** discounted

  - > Coupons **cannot** be applied to core fees

  - > Core fee **must always** be added manually

  - > Core fee **must always** appear as a separate line item

  - > Core fee **must have** the same cost and unit price

  - > Core fee **must be** added for the same quantity

![](/tmp/_media_ignore/media/image12.png)  
If any of these rules are broken, the order is **non-compliant**.

## **3. Pre-Sale Compliance (Before You Place the Order)**

### **Step 1: Product Verification**

Before quoting or placing the order, confirm:

  - > The **SKU is flagged as “Core Fee Applicable”**

![](/tmp/_media_ignore/media/image7.png)**  
**

  - > The **core fee amount matches exactly** what is listed for that SKU

  - > You understand **what part must be returned** as the core

If the SKU shows a core fee and it’s missing from your quote — **STOP. Fix it before proceeding.**

### **Step 2: Mandatory Customer Disclosure (You Must Say This)**

The customer **must clearly understand** all of the following **before checkout**:

You must communicate:

  - > Item price

  - > Core fee amount (**clearly stated as separate**)

  - > That the core fee is **refundable only after return  
    > **

  - > That **customer is responsible for returning the core  
    > **

  - > That a **prepaid return label will be provided  
    > **

There is no shortcut here. If the customer later disputes the fee and this wasn’t disclosed, **the company loses**.

### **Step 3: Documentation (Required for Audit Protection)**

You must document proof of disclosure using **at least one** of the following (preferably more):

  - > Call notes summary

  - > Email confirmation copy

  - > Chat transcript

![](/tmp/_media_ignore/media/image4.png)

If it’s not documented, it’s considered **not disclosed**.

## **4. Order Entry Standards (Critical Section)**

### **4.1 Quote / Order Structure**

Every compliant order **must look like this**:

**Line 1:** Product price  
**Line 2:** Core fee (separate line item)

### **4.2 Mandatory Core Fee Description (Must Be Copied Exactly)**

The **core fee line item must include this description**:

> *“PACKAGE WILL INCLUDE A PREPAID LABEL TO RETURN CORES.  
> PLEASE FOLLOW INSTRUCTIONS AND NOTIFY US WHEN CORES ARE RETURNED.  
> ONCE CORE(S) ARE RETURNED, CORE FEE WILL BE FULLY REFUNDED.”*
> 
> ![](/tmp/_media_ignore/media/image2.png)

Do **not** paraphrase. Do **not** shorten. This protects DKHardware.

## **5. Strict Prohibitions (Do NOT Do This)**

These actions are compliance violations:

  - > Bundling core fee with item price

  - > Discounting the core fee

  - > Applying coupons to the core fee

  - > Hiding or failing to disclose the core fee

  - > Assuming refunds are automatic

Any of the above can lead to **refund losses, chargebacks, or audit failure**.
