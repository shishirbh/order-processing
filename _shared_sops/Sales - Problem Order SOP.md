**<span class="underline">SOP TO WORK ON NEW ORDERS WITH PROBLEMS - SALES</span>**

# **PURPOSE**

This SOP defines the standardized process for handling orders placed under the status **“New Order with Problems – Sales.”**

These orders cannot proceed to fulfillment due to issues such as:

  - > Vendor pricing confirmation required

  - > Special order confirmation

  - > Customer approval needed

  - > Product discontinuation

  - > Long lead times

  - > Vendor sourcing issues

  - > Marketplace communication limitations

This SOP ensures:

  - > Consistent resolution

  - > Proper vendor/customer communication

  - > Complete documentation

  - > Correct status updates

  - > Compliance with internal order governance

# **SCOPE**

Applies to **all orders placed under the status “New Order with Problems – Sales.”**

Order sources include:

  - > DK Hardware website

  - > Marketplace orders (Amazon)

  - > Sales representative orders

  - > Manual orders placed through sales team

# **A. DETAILED WRITTEN SOP**

# **STEP 1 — Identify the Problem Type**

Open the order and determine which scenario applies.

Possible scenarios:

1.  > Vendor Quote Required

2.  > Customer Signed Quote Required

3.  > Vendor Quote + Customer Signed Quote Required

4.  > Discontinued Item

5.  > High Lead Time

6.  > Vendor Cannot Source Item

7.  > Marketplace Communication Restrictions (Amazon)

If **multiple scenarios apply**, complete **all applicable processes before moving order forward.**

# **STEP 2 — Vendor Quote Required**

## **When Vendor Quote is Required**

Vendor quote must be requested when:

  - > Large quantity order

  - > Bulk pricing verification required

  - > Pricing unavailable or unclear

  - > Custom / special order item

  - > Pricing deviation suspected

## **Action Steps**

1.  > Send quote request to vendor.

2.  > Wait for vendor response.

3.  > If no response → follow vendor follow-up protocol.

4.  > Upon receiving quote → verify details.

5.  > Adjust order pricing if needed.

6.  > Inform customer if price or lead time changed.

7.  > Attach vendor quote to order.

8.  > Add notes.

9.  > Change order status → **New Order  
    > **

## **Vendor Quote Verification Checklist**

Verify the following before proceeding:

| **Item**               | **Required Verification**  |
| ---------------------- | -------------------------- |
| SKU / Item Number      | Matches order              |
| Quantity               | Matches requested quantity |
| Unit Price             | Confirm pricing            |
| Bulk Price             | If applicable              |
| Ship Method            | Warehouse or Dropship      |
| Lead Time              | Vendor estimated           |
| Minimum Order Quantity | If applicable              |
| Total Price            | Accurate                   |
| Vendor Notes           | Any restrictions           |

# **STEP 3 — Customer Signed Quote Required**

## **When Signed Quote is Required**

Signed quote required if:

  - > Special order item

  - > Custom order

  - > Order value **\> $15,000  
    > **

  - > Vendor marks item **Non-Returnable / Non-Cancellable  
    > **

## **Action Steps**

1.  > Call the customer and explain:
    
      - > Item is non-returnable
    
      - > Item is non-cancellable

2.  > Create quote with exact order details.

3.  > Add text under item name:

\*\*\*\*NON CANCELLABLE / NON RETURNABLE\*\*\*\*

4.  > Review quote for accuracy.

5.  > Send quote to customer.

6.  > Wait for signed confirmation.

7.  > Follow up if customer does not respond.

8.  > Attach signed quote to order.

9.  > Change order status → **New Order  
    > **

## **Customer Signed Quote Email in case call not answered**

Subject: **Signed Quote Required to Process Your Order**

Dear (Customer Name),

I tried contacting you regarding Order \#\_\_\_\_ for (Item Name).

This order contains items that are **non-cancellable and non-returnable**, therefore we require a signed quote confirming your agreement before processing.

Please review, sign, and return the attached quote.

Once received, we will proceed with processing your order.

If you have any questions, please feel free to reply or call.

Best Regards  
(Rep Name)

# **STEP 4 — Vendor Quote + Customer Signed Quote Required**

Occurs when **both conditions apply simultaneously**, such as:

  - > Special order item with unknown pricing

  - > Custom item with vendor lead time

  - > Large special order

## **Action Steps**

1.  > Request vendor quote

2.  > Verify vendor quote

3.  > Create customer quote

4.  > Send for customer signature

5.  > Receive signed quote

6.  > Attach both documents to order

7.  > Add notes

8.  > Change order status → **New Order  
    > **

# **STEP 5 — Discontinued Item**

## **When Item is Discontinued**

Item is no longer manufactured or stocked by vendor.

## **Actions**

1.  > Verify with supplier if item is truly discontinued.

2.  > Ask vendor if replacement item exists.

3.  > Research internal alternatives.

4.  > Offer alternative to customer.

5.  > If customer approves → update order.

6.  > If customer declines → request cancellation.

7.  > Add notes.

## **Alternative Product Email**

(Rep Name)

# **STEP 6 — High Lead Time**

When vendor confirms **extended lead time.**

## **Actions**

1.  > Inform customer of lead time.

2.  > Research faster alternatives.

3.  > Offer replacement option.

4.  > If customer approves → update order.

5.  > If customer waits → leave order unchanged.

6.  > Document customer choice.

# **STEP 7 — Vendor Cannot Source Item**

Occurs when item cannot be obtained from supplier network.

## **Actions**

1.  > Confirm sourcing limitation.

2.  > Search alternative vendor.

3.  > If unavailable → offer alternative product.

4.  > If customer approves → update order.

5.  > If declined → request cancellation.

# **STEP 8 — Marketplace Orders (Amazon)**

Amazon masks contact details.

## **Key Rules**

  - > Email appears masked but **is valid for communication  
    > **

  - > Phone number may expire

  - > Contact window limited

  - > Must attempt communication within timeframe. Follow the steps as per the scenario

## **Communication Timeline**

| **Day** | **Action**                                    |
| ------- | --------------------------------------------- |
| Day 1   | Initial email                                 |
| Day 2   | First follow-up                               |
| Day 4   | Second follow-up                              |
| Day 6   | Final reminder                                |
| Day 7   | Add conclusive notes and request cancellation |

# **Documentation Requirements**

Always record:

  - > Vendor communication

  - > Customer communication

  - > Decision taken

  - > Alternatives offered

  - > Attachments added

# **Always Avoid**

  - > Processing special order without signed quote

  - > Changing order pricing without customer confirmation

  - > Skipping vendor quote verification

  - > Cancelling marketplace orders before contact attempts

# **B. DECISION MATRIX**

| **Scenario**            | **Trigger Condition**            | **Required Action**           | **Customer Contact**    | **Final Status**            |
| ----------------------- | -------------------------------- | ----------------------------- | ----------------------- | --------------------------- |
| Vendor Quote Needed     | Bulk order / unclear pricing     | Request vendor quote          | Notify if price changes | New Order                   |
| Customer Signed Quote   | Special order / \>$15k / NRNC    | Send quote for signature      | Mandatory               | New Order                   |
| Vendor + Customer Quote | Special order + vendor quote     | Obtain both                   | Mandatory               | New Order                   |
| Discontinued Item       | Vendor confirms discontinued     | Offer alternative             | Yes                     | New Order or Cancellation   |
| High Lead Time          | Vendor lead time extended        | Offer faster alternative      | Yes                     | New Order                   |
| Cannot Source           | Vendor cannot supply             | Offer alternative             | Yes                     | New Order or Cancellation   |
| Amazon Communication    | Marketplace contact restrictions | Follow communication schedule | Yes                     | Cancellation if no response |

# **D. QUICK RULE SHEET (CHEAT SHEET)**

| **IF**                        | **DO**                                     |
| ----------------------------- | ------------------------------------------ |
| Bulk quantity order           | Request vendor quote                       |
| Pricing unclear               | Request vendor quote                       |
| Special order item            | Get signed customer quote                  |
| Order \> $15k                 | Get signed customer quote                  |
| NRNC item                     | Signed quote required                      |
| Vendor quote + special order  | Get vendor quote AND customer signed quote |
| Item discontinued             | Offer alternative                          |

# **Escalation Rule**

If the situation is **not covered in this SOP**, consult your **Supervisor immediately before proceeding.**
